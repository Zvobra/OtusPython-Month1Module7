# Телефонный справочник

Консольное приложение на Python для ведения простого телефонного справочника. Программа хранит контакты в JSON-файле, позволяет просматривать список с пагинацией, искать, добавлять, редактировать, удалять и сохранять контакты.

## Возможности

- загрузка контактов из JSON-файла
- автоматическое создание файла контактов, если он отсутствует
- просмотр контактов в табличном виде
- пагинация списка по 10 контактов на страницу
- поиск по имени, телефону и комментарию
- добавление новых контактов
- выбор контакта по ID
- редактирование контактов
- удаление контактов
- сохранение изменений в файл
- запрос сохранения при выходе, если есть несохраненные изменения

## Требования

- Python 3.10 или новее.

Внешние зависимости не требуются: приложение использует только стандартную библиотеку Python.

## Запуск

```bash
python main.py
```

При запуске программа спросит путь к файлу контактов:

```text
Файл контактов [contacts.json]:
```

Можно нажать `Enter`, чтобы использовать файл по умолчанию `contacts.json`, или указать другой путь к JSON-файлу.

Если указать папку, приложение создаст или откроет в ней файл `contacts.json`. Если указанный файл не существует, он будет создан автоматически.

## Главное меню

После загрузки данных доступны действия:

1. `Показать все контакты` - открыть список контактов с пагинацией.
2. `Найти контакт` - найти контакты по имени, телефону или комментарию.
3. `Добавить контакт` - создать новый контакт.
4. `Выбрать контакт` - открыть контакт по ID для просмотра, редактирования или удаления.
5. `Сохранить` - записать текущие изменения в JSON-файл.
6. `Выход` - завершить работу приложения.

Внутри списка контактов можно перейти на следующую или предыдущую страницу, выбрать конкретную страницу, выбрать контакт по ID или вернуться назад.

## Формат данных

Контакты хранятся в JSON-файле в виде массива объектов:

```json
[
  {
    "id": 1,
    "name": "Ivan Petrov",
    "phone": "+79991234567",
    "comment": "Test contact"
  }
]
```

Поля контакта:

- `id` - положительное целое число, уникальный идентификатор контакта;
- `name` - непустая строка с именем;
- `phone` - непустая строка с телефоном;
- `comment` - строка с комментарием, может быть пустой.

При добавлении нового контакта ID назначается автоматически: берется максимальный существующий ID и увеличивается на 1.

## Сохранение

Изменения не записываются в файл сразу после добавления, редактирования или удаления. Чтобы сохранить данные, выберите пункт `Сохранить`.

При выходе приложение предупредит о несохраненных изменениях и предложит сохранить их.

## Примечания

- Файл контактов должен иметь расширение `.json`.
- В JSON-файле не должно быть дубликатов по `id`.
- Если JSON поврежден или имеет неверный формат, приложение завершится с сообщением об ошибке.
